# Packaged Bootstrap 5 HTML screens

Open `index.html` in a browser for the catalog. Each screen loads Bootstrap 5.3 from CDN plus `../tokens.css`.

| File | Purpose |
| --- | --- |
| `landing.html` | Marketing site — navbar, hero, features, pricing |
| `dashboard.html` | App shell — sidebar, stats, searchable table |
| `login.html` | Auth — centered sign-in card |
| `settings.html` | App — profile form + danger zone |
| `components.html` | UI kit — buttons, alerts, badges, modal |

Copy screens into `.larapilot/mockups/{spec}/` and adapt copy, colors, and routes per the PRD.
