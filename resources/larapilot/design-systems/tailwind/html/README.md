# Packaged Tailwind HTML screens

Open `index.html` in a browser for the catalog. Each screen loads Tailwind via CDN and uses **utility classes only**.

| File | Purpose |
| --- | --- |
| `landing.html` | Marketing site — navbar, hero, features, pricing, footer |
| `dashboard.html` | App shell — sidebar, stat cards, searchable table |
| `login.html` | Auth — centered sign-in card |
| `settings.html` | App — profile form + danger zone |
| `components.html` | UI kit — buttons, alerts, badges, form, modal preview |

Copy screens into `.larapilot/mockups/{spec}/` and adapt copy, `tailwind.config`, and routes per the PRD.
