# Filament — static HTML screens

Packaged HTML merged from **two** Figma community kits (see `../figma-sources.md`):

| Figma kit | URL |
| --- | --- |
| Design System | [1413822581847485668](https://www.figma.com/community/file/1413822581847485668/filament-3-design-system) |
| UI Kit (Free) | [1417716904167561805](https://www.figma.com/community/file/1417716904167561805/filament-3-free) |

Open `index.html` locally after `larapilot:install`.

| File | Frame |
| --- | --- |
| `dashboard.html` | Dashboard + stat widgets |
| `dashboard-dark.html` | Dashboard (dark mode) |
| `login.html` | Panel sign-in |
| `resource-list.html` | Resource index |
| `resource-create.html` | Create form |
| `resource-edit.html` | Edit form |
| `resource-view.html` | View / infolist |
| `settings.html` | Settings (tabs) |
| `empty-state.html` | Empty table |
| `modal.html` | Confirmation modal |
| `notifications.html` | Toast stack |
