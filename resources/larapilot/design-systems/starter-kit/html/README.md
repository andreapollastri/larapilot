# Laravel Starter Kits — static HTML screens

Packaged mockup catalog for Elise. Tokens from official [react-starter-kit](https://github.com/laravel/react-starter-kit) `app.css` — see `../sources.md`.

Open `index.html` locally in a browser.

| Screen | Kit mapping |
| --- | --- |
| `dashboard.html` | Default sidebar layout + placeholder content |
| `dashboard-dark.html` | Dark mode variant |
| `header-layout.html` | Header nav layout variant |
| `login.html` | Simple auth card |
| `auth-split.html` | Split auth layout |
| `settings.html` | Profile / password / appearance |
| `components.html` | Buttons, inputs, badges |
