# Packaged AdminLTE HTML screens

Open `index.html` in a browser for the catalog. Each screen loads AdminLTE 4 + Bootstrap 5.3 from jsDelivr CDN.

| File | Purpose |
| --- | --- |
| `dashboard.html` | App shell — dark sidebar, small-box widgets, orders table |
| `resource-list.html` | CRUD index — search toolbar, striped table, pagination |
| `login.html` | Auth — `login-box` with input groups |
| `settings.html` | Profile card + danger zone card |
| `components.html` | Info boxes, alerts, badges, collapsible card tools |

Copy screens into `.larapilot/mockups/{spec}/` and adapt copy, skins, and routes per the PRD.

Reference: [adminlte.io](https://adminlte.io/) · [AdminLTE 4 docs](https://adminlte-v4.netlify.app/docs/getting-started)
