# Client Materials

Place here any documentation, analysis, briefs, wireframes, API specs, spreadsheets, or other materials the client provides **before or during** discovery.

Larapilot skills **always** read and honor this folder alongside the PRD.

## Suggested contents

- Product briefs and requirement documents
- Existing analysis or audit reports
- Wireframes, mockups, or design exports
- API documentation (OpenAPI, Postman collections)
- Sample data files (CSV, JSON — no secrets)
- Meeting notes or stakeholder summaries

## Optional

Add an `INDEX.md` listing files and what each covers when the set is large.

## Privacy

Do not commit credentials, production dumps with personal data, or licensed third-party content without permission.
